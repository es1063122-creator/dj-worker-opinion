초보 설치가이드

      
1) 구글시트 만들기
최초 설정에서 ① 구글시트 만들기를 누르고 제목을 동진_종사자의견청취_관리대장으로 바꿉니다. 주소에서 /d/ 와 /edit 사이만 복사합니다. 이것이 SHEET_ID입니다.

      
2) Apps Script 만들기
② Apps Script 만들기를 누릅니다. 기본 Code.gs 내용을 모두 지웁니다.

      
3) 설치코드 붙여넣기
이 화면에서 ③ 설치코드 복사를 누르고 Apps Script의 Code.gs에 붙여넣습니다.

      
4) SHEET_ID 입력
코드 맨 위 const SHEET_ID = "여기에_구글시트_ID_입력"; 부분에 구글시트 ID만 넣습니다. 주소 전체를 넣으면 안 됩니다.

      
5) setupSheets() 실행
Apps Script 상단 함수 선택에서 setupSheets 선택 후 실행합니다. 권한 승인 후 구글시트에 의견접수, 반기점검 시트가 생기면 정상입니다.

      
6) 웹앱 배포 URL 저장
Apps Script에서 배포 → 새 배포 → 웹 앱을 선택합니다. 실행 사용자: 나, 액세스 권한: 모든 사용자로 배포합니다. 나온 URL을 이 관리화면 상단의 Google Apps Script 웹앱 URL에 저장합니다.

      
7) 테스트
단일 QR 안내문 화면에서 QR을 생성하고 제출화면을 열어 의견 1건을 넣습니다. 그 다음 본사 관리자에서 전체 새로고침을 눌러 접수 내용을 확인합니다.
